# medicalbooking
Build a modern digital platform that serves as a "touchpoint" connecting customers to the healthcare ecosystem. The website not only provides information but also integrates technological features to optimize the healthcare experience.

## Compile and run the project

### Backend Server

```bash
$ cd server
$ npm run build

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Frontend Portal

```bash
$ cd apps/portal
$ npm run dev -- -p 3001
```

### Frontend Admin

```bash
$ cd apps/admin
$ npm run dev -- -p 3002
```
