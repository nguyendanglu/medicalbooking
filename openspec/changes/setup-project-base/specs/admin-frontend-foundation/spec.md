## ADDED Requirements

### Requirement: Next.js Admin Dashboard Setup
The system SHALL have a Next.js frontend initialized in `apps/admin` with TypeScript, Tailwind CSS, and App Router.

#### Scenario: Verify Admin Initialization
- **WHEN** the command `npm run dev` is executed in `apps/admin`
- **THEN** the Next.js server starts successfully and is accessible on a local port (e.g., 3002)
